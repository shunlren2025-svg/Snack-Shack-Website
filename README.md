# The Snack Shack

A static website for a student lunch-time snack club.

## Files

- `index.html` - page structure
- `styles.css` - layout and visual styles
- `script.js` - menu search, cart, checkout, and stock behavior

## Run Locally

Open `index.html` in a browser, or serve this folder with any static web server.

No backend, database, accounts, framework, or build step is required.
